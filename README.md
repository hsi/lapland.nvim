# lapland.vim
Neovim/Vim theme inspired by Lapland's fruits

Features
--------
* Easy to read: more important elements have higher contrast, others are kept in the background.
* Using pleasing colors to the eye.
* Based on palette with 16 colors.
* Small and fast code.
